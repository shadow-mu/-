##### 3.幂函数的性质

- (x^p^)^q^ =x^p*q^   

  例子 （x^3^）^2^ =x^6^  != x ^5^

- X^p^ * X^q^ =x^p+q^

  例子（x^3^）^2^ =x^5^  != x^6^

- x^q^ \ x^p^ = x^p-q^   

  例子  x^3^ \ x^2^ = x  

- x^p^ \ 1 = x^-p^  

  例子  x \ 1 = x^-1^      x^3^ \ 1 = x^-3^   

- ^m^√￣x^n^ = x^m\n^  

  例子  ^2^√￣x^1^ = x^2\1^   ^3^√￣x = x^3\1^     ^4^√￣x^3^ = x^4\3^
