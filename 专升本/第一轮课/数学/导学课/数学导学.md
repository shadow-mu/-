###### 幂函数的性质

1. (x^p^)^q^ =x^p*q^   

   例子 （x^3^）^2^ =x^6^  != x ^5^

2. X^p^ * X^q^ =x^p+q^

   例子（x^3^）^2^ =x^5^  != x^6^

3. x^q^ \ x^p^  = x^p-q^   

   例子  x^3^ \ x^2^ = x  

4. x^p^ \ 1 = x^-p^  

   例子  x \ 1 = x^-1^ （反比例函数）      x^3^ \ 1 = x^-3^   

5. ^m^√￣x^n^ = x^m\n^  

   例子  ^2^√￣x^1^ =√￣x = x^2\1^   ^3^√￣x = x^3\1^     ^4^√￣x^3^ = x^4\3^
